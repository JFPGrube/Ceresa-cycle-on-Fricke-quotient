# Ceresa-cycle-on-Fricke-quotient
Program to verify that the Ceresa cycle of the quotient of the modular curve X_0(p) by the Fricke involution is not torsion up to rational equivalence. In my master thesis it is explained how this program works.

The .gp file provides a complete implementation in PARI/GP for the shadow of the Hecke operator T_2. An example command for how this may be used can be found at the very end of the file. The .ipynb is an implementation in SageMath which works for many more Hecke operators (depending on the prime p), but it does not check whether the condition on the tangent spaces is satisfied!
